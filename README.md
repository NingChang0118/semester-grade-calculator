# Semester Grade Calculator

A reusable grade calculator for students to estimate required scores for finals, assignments, and projects.

## Motivation

Near the end of a semester, students often need to calculate how many points they need on remaining exams, assignments, or projects to reach a target final grade.

Manually creating spreadsheet formulas for every course is repetitive and error-prone, so this project aims to provide a reusable grade calculation tool.
